jQuery.pager.js

Automatically paginate any type of content using javascript (build on top of jQuery)

Example

This is my custom example for which I needed to build this plugin.


  
  $('#demo-1').pager({
    items_per_page: 6,
    items: '.item',
    custom_control: [3,6,9]
  });
